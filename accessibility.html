Accessibility

Everyone deserves a seat at the table — online too.

Hummus Republic is committed to making our website usable for as many people as possible. This concept site was built with accessibility in mind — keyboard-navigable controls, visible focus states, reduced-motion support, semantic structure and color contrast aligned with WCAG 2.1 AA.

Ongoing work

Accessibility is never finished. We continue to test with assistive technologies and welcome feedback that helps us improve.

Need help or spot an issue?

If you have trouble accessing any part of this site, please let us know and we'll help and fix it.
